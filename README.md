# In-Memory File System

### API

1. create_root()
1. file_exists()
1. file_type()
1. ls()
1. mkdir()
1. touch(): create file
1. write(): write data to file, create if it doesn't exist
1. read(): read contents of file
1. cat(): concatenate contents to file
1. mv(): also used for renaming
1. cp()
